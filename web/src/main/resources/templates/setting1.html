KPI Management Software User Guide The KPI management software helps you efficiently track and manage plans and tasks. Below is a detailed guide on how to use the features of the software:

1. Managing Plans

Creating a New Plan

  1. Go to the "Plans" page.
  2. Click the "Create Plan" button.
  3. Enter the plan name, description, and other relevant information.
  4. Click "Save" to create a new plan.

Viewing and Editing a Plan

  1. Select the plan you want to view or edit from the plan list.
  2. Click on the plan name to open the plan details.
  3. Edit the plan information if needed and click "Save" to update.

2. Managing Tasks

Adding a New Task to a Plan

  1. Open the plan to which you want to add a task.
  2. Click the "Add Task" button.
  3. Enter the task name, description, status (Completed, In Progress, Not Started), deadline, priority, and progress.
  4. Click "Save" to create a new task.

Updating Task Status and Progress

  1. Select the task you want to update from the task list within the plan.
  2. Click on the task name to open the task details.
  3. Update the status, completion progress, and other information if needed.
  4. Click "Save" to save the changes.

3. Viewing History

  1. Go to the "History" page.
  2. View the list of recent activities by you and your team members.
  3. Click on individual activities to see details if needed.

4. Messaging and Attaching Files

Sending a Message

  1. Go to the "Messages" page.
  2. Select the recipient from your contact list or enter the recipient's name.
  3. Type your message content.
  4. Click "Send" to send the message.

Attaching Files or Images

  1. While composing a message, click on the attachment icon.
  2. Choose a file or image from your computer.
  3. Click "Attach" to add the file to your message.
  4. Click "Send" to send the message with the attached file.

5. Viewing the Schedule

Viewing the Weekly Schedule

  1. Go to the "Schedule" page.
  2. Select the weekly view to see your tasks for the current week.
  3. Click on each day to see the details of the tasks for that day.

Tracking Long-Term Tasks

  1. In the schedule, select the task that spans multiple days.
  2. View the highlighted or marked days to know the duration of the task.
  3. Click on the task to see details and edit if necessary.

Conclusion

The KPI management software provides various tools to help you manage your plans and tasks efficiently. Make sure to regularly update the status and progress of tasks to track them timely and achieve your set goals. If you encounter any issues, please contact the technical support team for assistance. Happy using the software!
